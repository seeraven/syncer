Conversion to ICO
-----------------

Use the convert_to_ico.sh script, which uses ImageMagick for the conversion
like the following example:

    convert syncer-warning-icon.png -define icon:auto-resize=256,128,64,48,32,16 syncer-warning-icon.ico
